django-codecompare

Stateless Django library for code diff, similarity scoring, and plagiarism detection. No database required for comparison — just POST and get results.

Python kodida to'g'ridan-to'g'ri ishlatish (API siz)
from codecompare.core.services import ComparisonService

svc = ComparisonService()
result = svc.compare(
    old_code="def add(a, b): return a + b",
    new_code="def sum(x, y): return x + y",
    language="python",
)

print(result.similarity.overall)          # 62.5
print(result.plagiarism.is_plagiarism_suspected)  # True/False
print(result.plagiarism.confidence)       # 87.3
print(result.statistics.renamed_symbols)  # [{"old_name":"add","new_name":"sum",...}]

Javob (Response) — HTTP 200
similarity — O'xshashlik ko'rsatkichlari (%)
similarity.overallfloatUmumiy o'xshashlik — barcha usullar o'rtacha qiymati
similarity.exactfloatBelma-bel to'liq moslik (Jaccard)
similarity.tokenfloatToken darajasidagi o'xshashlik
similarity.levenshteinfloatLevenshtein/Wagner-Fischer masofasi
similarity.cosinefloatTF-IDF cosine o'xshashlik
similarity.astfloatSintaksis daraxt (AST) tuzilmaviy o'xshashlik
similarity.semanticfloatWinnowing fingerprint orqali semantik o'xshashlik
plagiarism — Ko'chirmachilik ko'rsatkichlari
plagiarism.is_plagiarism_suspectedboolKo'chirmachilik gumonlanadimi? (confidence > 70% bo'lsa true)
plagiarism.confidencefloatKo'chirmachilik ehtimoli (0–100%)
plagiarism.renamed_identifierslistQayta nomlangan o'zgaruvchi/funksiyalar: [{"old_name","new_name","confidence"}]
plagiarism.structural_fingerprint_matchfloatKod tuzilmasi (fingerprint) mos kelish foizi
plagiarism.whitespace_stripped_similarityfloatBo'shliqlar olib tashlanganda o'xshashlik
statistics — O'zgarish statistikasi
statistics.lines_addedintQo'shilgan qatorlar soni
statistics.lines_removedintO'chirilgan qatorlar soni
statistics.functions_addedlistYangi qo'shilgan funksiyalar nomlari
statistics.functions_removedlistO'chirilgan funksiyalar nomlari
statistics.renamed_symbolslistQayta nomlangan simbollar: [{"old_name","new_name","confidence"}]
statistics.complexity_deltaintSiklik murakkablik o'zgarishi (+ yoki -)
diff — Qator farqlari (Myers algoritmi)
diff.addedintJami qo'shilgan qatorlar
diff.removedintJami o'chirilgan qatorlar
diff.chunks[]listUnified diff bo'laklari (git diff formatiga o'xshash)
diff.chunks[].headerstringMisol: @@ -2,1 +2,1 @@
diff.chunks[].lines[].change_typestringequal | insert | delete
old_complexity / new_complexity — Kod murakkabligi
*.cyclomaticintSiklik murakkablik (McCabe) — past bo'lsa yaxshi (<10)
*.maintainability_indexfloatSaqlanish indeksi 0–100 (yuqori = yaxshi, <20 = muammo)
*.halstead_volumefloatHalstead hajmi — kod axborot miqdori
*.lines_of_codeintJami kod qatorlari (LOC)
Boshqa maydonlar
